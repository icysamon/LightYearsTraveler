# Light Years Traveler
This is a game for [Godot Wild Jam #71
](https://icysamon.itch.io/light-years-traveler).  

![](.//doc/readme.png)

## Controls
- **move** - left mouse
- **attack** - right mouse
- **zoom in** - scroll up
- **zoom out** - scroll down

Rest at the seasons planet to recharge your batteries.
