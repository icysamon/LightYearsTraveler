# Light Years Traveler
Godot Wild Jam #71 game.  

itch: https://icysamon.itch.io/light-years-traveler  

![](.//doc/readme.png)

## Tutorial
- **move** - left mouse
- **attack** - right mouse
- **zoom in** - scroll up
- **zoom out** - scroll down
Rest at the seasons planet to recharge your batteries.
